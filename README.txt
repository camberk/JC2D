Our website is an exercise tracking website that incentivizes users to exercise by 
allowing users to earn points based on exercise information entered and where the point 
amounts are compared to others and points can be used to unlock rewards.

External Libraries/Frameworks Used: bootstrap, Chart.js, Vue.js
